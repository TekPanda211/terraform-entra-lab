\# Governance Catalog



\## Purpose



The Governance Catalog defines standardized access offerings within BlackKnight One.



Each catalog represents a collection of governed resources that can be requested through Microsoft Entra ID Identity Governance.



\---



\# Workforce Access



Standard employee access.



\### Packages



\- New Hire Baseline

\- Department Access

\- Manager Access

\- Executive Access



\---



\# Privileged Access



Elevated administrative access.



\### Packages



\- Global Administrator

\- Privileged Role Administrator

\- Exchange Administrator

\- SharePoint Administrator

\- Teams Administrator

\- Security Administrator



\---



\# Contractor Access



Temporary access for external workers.



\### Packages



\- Standard Contractor

\- Vendor Support

\- Temporary Project

\- Seasonal Workforce



\---



\# Application Access



Application-specific access.



\### Packages



\- CRM

\- ERP

\- Finance

\- HR Systems

\- Engineering Tools



\---



\# Guest Collaboration



External collaboration.



\### Packages



\- Business Partner

\- Customer Collaboration

\- Legal

\- Audit
