\# v0.2.0 - Identity Architecture



This release expands the Terraform Entra Lab from basic identity foundation into a more structured IAM architecture.



\## Added



\- Environment-based naming using `company\_prefix` and `environment`

\- `terraform.tfvars` configuration for environment values

\- Stable Terraform `for\_each` keys for group management

\- Production naming pattern using `BlackKnight-Prod`

\- Role-assignable Microsoft Entra ID groups

\- Privileged group framework for future RBAC and PIM scenarios



\## Infrastructure Added



\- BlackKnight-Prod-IT-Admins-RoleAssignable

\- BlackKnight-Prod-Security-Admins-RoleAssignable

\- BlackKnight-Prod-Engineering-Admins-RoleAssignable



\## Updated



\- Group naming standard

\- Terraform variable strategy

\- Terraform locals strategy

\- Department group resource structure



\## Next Milestone



v0.3.0 will focus on Administrative Units, dynamic groups, and identity lifecycle design.
