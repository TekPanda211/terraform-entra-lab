\# License Operations Engine



\## Purpose



The License Operations Engine supports license readiness, optimization, and governance.



Licensing is part of identity operations because licensing controls access to features, security capabilities, and governance tools.



\## Capabilities



\- License optimization

\- Unlicensed user detection

\- Disabled users with active licenses

\- Duplicate license assignment review

\- Orphaned license detection

\- Group-based licensing validation

\- Premium feature readiness checks



\## Example Checks



\- Users with no assigned license

\- Disabled users with active licenses

\- Users assigned premium licenses without need

\- Users missing required licenses for governance features

\- Groups used for licensing with no owners

\- License assignment drift
